# SortingVisualizer
This is a visualization tool for various sorting algorithms
 - BubbleSort
 - SelectionSort
 - MergeSort
 - QuickSort
 - HeapSort


## Functionalities
- Array size can be changed
- Visualization speed can be adjusted on the fly
